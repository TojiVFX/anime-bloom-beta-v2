Push to Site
Make your changes, then click the button below. All 4 files get pushed to GitHub and Vercel auto-deploys your site in about 30 seconds.
Setup — Vercel Environment Variables
1
Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token. Select scope: repo. Copy the token.
2
Go to Vercel → Your Project → Settings → Environment Variables and add all four:

GITHUB_TOKEN = your token
GITHUB_OWNER = your exact GitHub username
GITHUB_REPO = your exact repo name
GITHUB_BRANCH = main or master
3
Redeploy once from Vercel dashboard so env vars take effect, then use Test Connection below to verify everything is working.
