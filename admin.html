Push to Site
Make changes then push. All files go to GitHub and Vercel auto-deploys in ~30 seconds.
Setup — Vercel Environment Variables
1
GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate. Scope: repo. Copy token.
2
Vercel → Project → Settings → Environment Variables: GITHUB_TOKEN, GITHUB_OWNER, GITHUB_REPO, GITHUB_BRANCH=main
3
Redeploy once from Vercel dashboard, then use Test Connection below.
